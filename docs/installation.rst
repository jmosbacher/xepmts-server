============
Installation
============


Stable release
--------------

To install xepmts-server, run this command in your terminal:

.. code-block:: console

    $ pip install xepmts_server

This is the preferred method to install xepmts-server, as it will always install the most recent stable release.

If you don't have `pip`_ installed, this `Python installation guide`_ can guide
you through the process.

.. _pip: https://pip.pypa.io
.. _Python installation guide: http://docs.python-guide.org/en/latest/starting/installation/


From source
-----------

The source for xepmts-server can be downloaded from the `Github repo`_.

You can either clone the public repository:

.. code-block:: console

    $ git clone git://github.com/jmosbacher/xepmts_server

Or download the `tarball`_:

.. code-block:: console

    $ curl -OJL https://github.com/jmosbacher/xepmts_server/tarball/master

Once you have a copy of the source, you can install it with:

.. code-block:: console

    $ pip install .

.. _Github repo: https://github.com/jmosbacher/xepmts_server
.. _tarball: https://github.com/jmosbacher/xepmts_server/tarball/master
